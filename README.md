# 💼 Personal Portfolio Website
A modern and responsive personal portfolio website showcasing my projects, technical skills, and software engineering journey.

## 🚀 Live Demo
🔗 **Portfolio:** [Access Link](https://personal-portfolio-two-lyart-g8ug956fme.vercel.app/)

## 📌 Features
* Responsive design for desktop and mobile
* Modern dark-themed UI
* Unique keyboard-inspired About section
* Technical skills showcase
* Featured project section
* Contact form integrated with Google Sheets
* Resume download option
* Social media links (GitHub, LinkedIn, LeetCode)

## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Apps Script (Contact Form)
* Vercel (Deployment)
